# Employee Management API

This project is a backend REST API built with Spring Boot and connected to a MySQL database. It supports CRUD operations for employee records.

## Tech Stack

- Java
- Spring Boot
- MySQL
- JDBC

## Features

- Retrieve all employees
- Retrieve an employee by ID
- Create a new employee
- Update an employee's salary
- Delete an employee by ID

## API Endpoints

### Get all employees
`GET /employees`

### Get employee by ID
`GET /employees/{id}`

### Create employee
`POST /employees`

Example request body:

```json
{
  "name": "Albert",
  "lastname": "Riemann",
  "dob": "1990-05-10",
  "salary": 3500.0
}
```

### Update salary
`PUT /employees/{id}/salary`

Example request body:

```json
{
  "salary": 5000.0
}
```

### Delete employee
`DELETE /employees/{id}`


## Database Setup

Create a MySQL database named `sysventa`:

```sql
CREATE DATABASE sysventa;
USE sysventa;
```

Then create the table:

```sql
CREATE TABLE empleado (
                          id INT AUTO_INCREMENT PRIMARY KEY,
                          nombre VARCHAR(100),
                          apellido VARCHAR(100),
                          fechaNacimiento DATE,
                          sueldo FLOAT
);
```

### Notes

- `fechaNacimiento` must follow the format `YYYY-MM-DD`
- The `id` field is auto-generated by the database
## How to Run

1. Clone the repository.
2. Create a MySQL database named `sysventa` and create the `empleado` table as described in the Database Setup section.
3. Copy `src/main/resources/application-example.properties` to `src/main/resources/application.properties`.
4. Update the database credentials in `application.properties`.
5. Run the Spring Boot application.

## Notes

- The API uses JDBC for database interaction.
- The project follows a simple DAO pattern.
